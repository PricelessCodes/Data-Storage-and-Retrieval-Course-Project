من ويكيبيديا، الموسوعة الحرة

< قالب:!!


This is an auxiliary template that allows one to encode "|" (pipe or vertical bar) within template parameters. If it is unnecessary to get a "|", it is better to use "&#124;". However the "|" delimiter for a Wiki table has to be escaped by a template if it is used within a template.

[تحرير] Notes

There is no such problem with "!" (exclamation mark). This problem only affects "|" (vertical bar) in conjunction with the m:Help:Parameter default mechanism within tables.

This template cannot be substituted.
